# EcommerceShop - ASP.NET Core MVC Project

### Giới thiệu
Đây là dự án Website cá nhân được xây dựng nhằm mục đích thực hành các kỹ năng lập trình với .NET.

### Công nghệ sử dụng
- **Backend:** .NET 10, Dapper, MSSQL Server
- **Frontend:** Bootstrap 5, JQuery
- **Khác:** Git, MVC Pattern

### Tính năng
- **Quản lý sản phẩm:** Xem chi tiết, lọc sản phẩm; Thêm, sửa, xoá sản phẩm.
- **Giỏ hàng:** Thêm / Xóa sản phẩm, tính toán giá tiền đơn hàng.
- **Hoá đơn:** Xem hoá đơn với các sản phẩm và giá tiền.
- **Admin trong Dashboard:** Xem thông tin của các user.
- **Role trong Dashboard:** Xem các role phân quyền của email.
- **Các trang API**: Xem các dữ liệu JSON của user, role, auth.
